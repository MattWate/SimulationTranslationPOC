Simulation Translation POC

Live translated narration for learning videos

Select English, isiZulu or Afrikaans. The POC translates the script on demand, mutes the original video when needed, reads the translated narration aloud and pauses playback if narration needs time to catch up.

POC rules

  • English uses the original video audio and English captions.
  • isiZulu and Afrikaans mute the original video and use translated narration.
  • Translations are generated when the language is selected, then cached in this browser session.
  • If browser voice support is limited, captions still show the translated text.
